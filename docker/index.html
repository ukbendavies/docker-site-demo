    • Be sure to check the add python to path box!

    Publish the site for production¶

    Create a Dockerfile

    • Take latest nginx image

    • Build out the new docker image layer

      • Set working directory inside the image
      • Copy the mkdocs output: site into the image
    1
2
3
    FROM nginx
WORKINGDIR /usr/share/nginx/html
COPY site .

    Build the doc site and the new docker image¶

2

docker build --tag mkdocs:v1 .
